# Quiz Game

A lightweight, browser-based multiple-choice Quiz Game built with plain HTML, CSS, and JavaScript.

![Main screen](./Screenshot%202024-11-04%20174145.png)

![Question screen](./Screenshot%202024-11-04%20174200.png)

## About

This small project presents a series of multiple-choice questions, tracks the player's score locally in the browser, and provides a clean interface for practicing or testing knowledge. It was created to be easy to read, modify, and extend.

## Features

- Multiple-choice questions
- Score tracking
- Keyboard and mouse-friendly controls
- No build tools or dependencies — works in any modern browser

## Demo

Open `index.html` in a browser to try the game. The screenshots above show the main screen and a sample question screen.

## Getting started

1. Clone the repository:

```bash
git clone https://github.com/BinaryVortex/Quiz-Game-2.git
```

2. Open `index.html` in your browser (double-click or use your browser's Open File option).

No additional setup is required.

## Project structure

- `index.html` — Main HTML file
- `style.css` — Styles for the game UI
- `script.js` — Game logic and question data
- `Screenshot 2024-11-04 174145.png` — Main screen image
- `Screenshot 2024-11-04 174200.png` — Question screen image

If your filenames differ, update the image paths or file names accordingly.

## Customize

- To add or edit questions, open `script.js` and modify the questions array.
- To change the look-and-feel, edit `style.css`.
